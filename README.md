# 🛒 PureBasket

PureBasket is a modern grocery shopping mobile application designed to provide a seamless and convenient online grocery shopping experience. Users can browse fresh organic products, view detailed product information, manage their cart, and place orders effortlessly.

## ✨ Features

- 🌿 Beautiful onboarding and splash screens
- 👤 User registration and authentication
- 🛍 Browse organic fruits and vegetables
- ⭐ Product ratings and reviews
- 📦 Product detail pages
- ➕ Add-to-cart functionality
- 🛒 Cart management
- 📱 Clean and responsive UI
- 🎨 Modern grocery-themed design
